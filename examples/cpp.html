Data types

Let's write a program to print hello, world!

// Print, Hello, world to console!
#include <iostream>

int main()
{
    std::cout << "Hello, world!";
    return 0;
}

What's the fun in a program that prints the same output all the time? Let's make it get an input and print it.

// Program that gets a name, and prints it to the console
#include <iostream>
#include <string>

int main()
{
  std::string name;
  std::cout << "What is your name? ";
  getline (std::cin, name);
  std::cout << "Hello, " << name << "!\n";
  return 0;
}
